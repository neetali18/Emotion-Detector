# Emotion Detector

## Project Overview
Emotion Detector is an AI-based web application developed using Watson NLP and Flask. The application detects emotions from user text input.

## Features
- Emotion detection
- Flask web application
- Error handling
- Unit testing
- Static code analysis

## Technologies Used
- Python
- Flask
- Watson NLP
